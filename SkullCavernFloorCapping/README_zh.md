# Skull Cavern Floor Capping

添加了新的机制，妈妈再也不用担心跳入竖井时跳过骷髅洞穴 200 层和 300 层的宝藏房间了。

例如，在 199 层跳入竖井时，本模组会确保你在第 200 层落地，而不是直接跳过 200 层。
