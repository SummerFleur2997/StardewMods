# SummerFleur's Better Hats

简体中文 | [English](README.md)

在星露谷的原版游戏中，帽子往往只是昂贵的装饰品——解锁成就后购买它们，除了作为一个装饰品以外，就别无他用，这似乎有些鸡肋了。因此我制作了这个模组，为游戏中几乎所有的帽子新增了独一无二的功能和效果。

本模组包含的帽子功能主要分为四大类：

1. 实用增益：提供各种各样的 Buff，助力你的 627 生活。
2. 机制改变：修改游戏机制，带来意想不到的便利或全新玩法。
3. 社交互动：佩戴特定帽子与可结婚的村民聊天，可以触发特殊的对话，并获得额外的友谊值。
4. 趣味整活：纯粹为了整活儿，为生活增添一丝幽默与惊喜。

希望这个模组能让你的星露谷之旅更加丰富多彩，享受每一顶帽子带来的乐趣！

本模组需要 [Better Hats API (BHA)](../BetterHatsAPI) 作为前置，而 BHA 提供了一个方便强大的帽子图鉴菜单，你可以在这个菜单里了解所有帽子的效果及其详细描述（菜单快捷键默认为 `Ctrl + Alt + H`）
